Local server · four pipelines

Web Summarizer

Summarize pages, podcasts, long-form video, and meeting bundles in one place. Credentials live in config.json on the machine that runs this service.

Workspace

Pick a module — one task runs at a time on the server.

01 · Web

Web summarization

Edge fetches the article body; the model returns summary and tags. Images stay under the task folder on disk; Markdown uses relative paths.

Supported: Zhihu, Xiaohongshu, WeChat articles, generic pages.

02 · Audio

Audio & podcasts

Download audio, transcribe, then summarize. Paste the show or episode page URL from the provider.

Supported: Ximalaya, Xiaoyuzhou (Cosmos), NetEase Cloud podcast pages.

03 · Video

Video analysis

Same pipeline as video_main.py: slide frames, transcript, alignment, combined report. Large files take longer.

Formats: mp4, mkv, mov, webm, avi, m4v. Sample rate follows the server default.

04 · Meeting

Meeting notes

Same flow as meeting_main.py: zip recording + timestamped screenshots (one folder layout), then upload the .zip.
